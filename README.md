## Overview

Thunder Executor is a Windows-native Roblox script injection utility built to load and run Lua scripts inside the Roblox client from a simple desktop interface. It is designed to stay lightweight while still giving access to an integrated script hub with hundreds of community scripts, ranging from basic automation to more advanced game edits.

The workflow is kept short with one-click injection. After launch, the app looks for an active Roblox session and gets the Lua execution environment ready automatically. Its bundled script library refreshes on its own, so new content appears without manual fetching. Thunder Executor is tuned for Roblox builds in 2026 and does not use activation keys or third-party account sign-ins.

## FAQ

**Is Thunder Executor safe to use?**  
This tool changes the Roblox client process while it is running. Use it at your own discretion and follow Roblox's Terms of Service.

**Does it work with the latest Roblox update?**  
Yes. The executor updates itself automatically to stay aligned with Roblox's 2026 builds.

**How does it compare to other script executors?**  
Thunder Executor is built around a low footprint and a large built-in script hub, which helps reduce reliance on outside script sources.

**Can my Roblox account be banned?**  
Third-party executors are against Roblox's rules. Account enforcement is handled by Roblox, and this tool does not promise account protection.

**Where are scripts stored locally?**  
User-added scripts are kept in the `scripts/user/` folder inside the installation directory.
